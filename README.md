# bookshare
BookShare - Plateforme cloud de partage/location de manuels scolaires. Choix : partage gratuit ou location payante. Architecture distribuée Azure (Cosmos DB, Kubernetes, ML). Scalable, fault-tolerant, collaborative. Paiement mobile money, géolocalisation, chat temps réel. Impact : accès à l'éducation pour familles démunies au Cameroun.
